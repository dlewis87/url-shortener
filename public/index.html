URL Shortener Microservice

Example creation usage:

http://stark-garden-69362.herokuapp.com/new/www.google.com

Example creation output

{ "original_url":"www.google.com", "short_url":"http://stark-garden-69362.herokuapp.com/8170" }

Usage:

http://stark-garden-69362.herokuapp.com/8170

Will redirect to:

https://www.google.com/
