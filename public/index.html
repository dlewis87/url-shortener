URL Shortener Microservice

Example creation usage:

http://stark-garden-69362.herokuapp.com/new/https://www.google.com

http://stark-garden-69362.herokuapp.com/new/http://foo.com:80

Example creation output

{ "original_url":"http://foo.com:80", "short_url":"http://stark-garden-69362.herokuapp.com/8170" }

Usage:

http://stark-garden-69362.herokuapp.com/2871

Will redirect to:

https://www.google.com/
